# Плагины интерфейса :
Автор оригинального плагина интерфейса является автор https://t.me/KakEgorka . Оригинал делает интерфейс красивым (постеры становятся горизонтальными ,подгружаются логотипы ). 
В данные плагины интерирован код ув. https://t.me/ipavlin98 с его разрешения (http://honeyxcat.github.io/lampa-logo/lampa-logo.js).
Что было изменено : 
Доступно две версии горизонтальные и вертикальные постеры на страницах. Интегрированы настройки в пункт Интерфейс из плагина lampa-logo.js. Настройки влияют на все доступные логотипы,дополнтиельно была добавлена настройка 
высоты логотипов (меняется если есть в tmdb).
В варианте горизонтальные постеры лого также наклывадется и на маленькие карточки,вместо названий снизу.
В обоих вариантах описание, жанр, год, страна производвтва и плашка TMDB были перенесена в правую часть (так мы избегаем  в 99% случаях "съядания" описания,если оно длинное.
В варианте с вертикальными постерами добавлена возможность вкл или откл подписи снизу карточек.
